<h1>Reporta Ya – Aplicación Web para Sevilla</h1>

<p>
  “Reporta Ya” es una aplicación web para Sevilla que permite a los vecinos
  reportar incidencias urbanas —como baches, farolas averiadas, basura o ruido—
  añadiendo foto, descripción y ubicación. El Ayuntamiento y las comunidades
  reciben los avisos, los gestionan desde un panel administrativo y pueden
  marcar las incidencias como resueltas. Incluye mapa interactivo, filtros por
  categoría y estado, estadísticas y un diseño responsive con la identidad
  visual indicada.
</p>
